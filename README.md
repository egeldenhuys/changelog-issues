# changelog-issues
Generate links for issues referenced in a changelog

## Usage
```
./generate.sh <CHANGELOG_PATH> <USERNAME/REPO>

$ ./generate.sh /path/to/CHANGELOG.md TripleParity/docks-ui

[#131]: https://github.com/TripleParity/docks-ui/issues/131
```
